# Hero Apps

##  Description
Hero Apps is a modern and high-performance React-based web application. It is built with the latest frontend technologies to deliver a fast, responsive, and visually appealing user experience.

The application focuses on clean UI design, smooth navigation, and efficient data handling.

---

##  Features
-  Fast and optimized performance
-  Beautiful and modern UI design
-  Fully responsive (Mobile, Tablet, Desktop)
-  Dynamic data handling with API
-  User-friendly notifications
-  Data visualization with charts

---

## Technologies Used

### Frontend
- React
- React DOM
- React Router & React Router DOM

### Styling
- Tailwind CSS
- @tailwindcss/vite

### Libraries
- Axios (API calls)
- React Icons (Icons)
- React Toastify (Notifications)
- Recharts (Charts)
